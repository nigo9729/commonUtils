## @kqfe/utils

凯桥信息通用工具库

### ✨ 特性

- @kqfe/utils 是一个工具库，提供了一些常用的工具函数。

- 纯js实现，无任何依赖，支持tree-shaking。

- 轻量化，安装快

### 📦 安装

```bash
# npm
npm install @kqfe/utils
# yarn
yarn add @kqfe/utils
# pnpm
pnpm install  @kqfe/utils

```

### 🔨 使用

```typescript
import { desensitizeName } from '@kqfe/utils';
```
